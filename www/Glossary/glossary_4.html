Usecase: Usecase short name

Form: ODK generated form used for the survey

Country: National administrative level. Target geography of the specified project.

Enumerator: Survey personnel charged with data collection

Household: An agricultural household where at least one member of the household is operating a holding or when the household head, reference person or main income earner is economically active in agriculture..

ENID: Enumerator identification number (Hover to get enumerator details).

HHID: Household identification number.

Complete: Submitted with correct information within required time period.

Missing Details: Missing information within required time period.

Overdue: Submission overdue with or without information submitted.

Future Event: Future data collection event, due date not yet passed and information not yet submitted.

Events: Various field events monitored (these can vary with usecase).

Site Selection: When the household was registered by enumerator. Obtained from Register or Verify Household form.

Event 1: Data collection event 1/ Planting date - When planting was done.

Event 3: Data collection event 3/ Field monitoring.

Event 4: Data collection event 4/ Field monitoring.

Event 5: Data collection event 5/ Field monitoring.

Event 6: Data collection event 6/ Field monitoring.

Event 7: Data collection event 7/ Harvest date - When harvesting was done.

Data Preview: Preview and download data as submitted via ODK forms by the enumerators
